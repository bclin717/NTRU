# Implement NTRU with Proxy Re-Encryption and Homomorphism in C

## Properties
  - Quantum Resistant
    - Based on Ring-LWE
    - NTRU
  - Homomorphism
    - Fully Homomorphic
    - Somewhat Homomorphic Encryption
        - YASHE(Yet Another Somewhat Homomorphic Encryption)
  - Proxy Re-Encryption
